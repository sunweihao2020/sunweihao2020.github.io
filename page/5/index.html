CESM PE layout

本文探讨核数设置如何影响CESM的运行

网上资料：

CESM user guide

网上文档

阅读文档

The component processor layout is via three settings: the number of MPI tasks, the number of OpenMP threads per task, and the root MPI processor number from the global set.

ONE IMPORTANT POINT is that for a fully active configuration, the atmosphere component is hardwired in the driver to never run concurrently with the land or ice component.

大气模块不能和land ice模块并行。

If all components have identical NTASKS, NTHRDS, and ROOTPE set, all components will run sequentially on the same hardware processors.

如果懒省事统一给一个值，那么是顺序挨个运行

以此为例，如果写提交脚本，那么总共需要申请128个 hardware processors

128核排列顺序为

Atmosphere：4 4 4 4 4 4…共计64

Ocean：1 1 1 1 1…共计64

mac设置免密登陆

mac系统中的iterm软件是可以实现自动ssh登录的，但是亲测iterm对于本地命令行支持的比较好，一旦连接到服务器上，在终端的显示上经常会出现BUG。而mac终端又没有类似的功能。因此这里实现的思想是 mac终端设置登录指令 + 服务器上设置ssh密钥实现免密。

  1. mac终端设置登录指令

在右侧添加服务器ip

这样下次登录的时候，右键点击终端，点击新建远程连接即可快速生成ssh 服务器指令

  1. mac与服务器设置ssh密钥

在 ~/.ssh 目录下 输入

在出来的这一行里面可以输入自己喜欢的名字，如果不输入名字的话就是默认的文件名，这可能会覆盖掉原有的，因此建议还是另取名，这里取名swh，然后按两次回车，即可生成密钥，ls后可见带.pub的是公钥

生成ssh密钥之后，将私钥加入ssh服务器

注意这一步加入是重要的，不然在服务器上设置好了也连不上

服务器上的设置

打开公钥文件 swh.pub ，复制了内容之后，ssh到服务器主机下 ~/.ssh 路径

把刚才复制的本地公钥内容复制进去退出即可。

mac ssh登陆

右键点击终端，点击新建远程连接，点击对应服务器ip即可快速免密登陆了～
